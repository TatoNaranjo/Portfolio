## Sobre este Portafolio :book:
Este repositorio contiene el código fuente de mi portafolio personal y profesional. Es una **Aplicación de Página Única (SPA)** estática, diseñada para ser extremadamente rápida, accesible y con un alto rendimiento SEO.

El proyecto sigue una arquitectura basada en componentes y una filosofía de diseño **Mobile-First**. Su propósito es mostrar mi experiencia como Ingeniero de Sistemas, mis proyectos destacados (incluyendo integraciones de IA y algoritmos genéticos) y mi trayectoria en la programación competitiva.

Actualmente desplegado y alojado en **Netlify**.

## Stack Tecnológico :warning:
Este proyecto fue construido utilizando las tecnologías más modernas del ecosistema web:

* **[Astro](https://astro.build/):** Framework principal para generar un sitio estático optimizado (SSG).
* **[React](https://react.dev/):** Utilizado mediante "Astro Islands" para componentes interactivos (como modales y menús).
* **[Tailwind CSS](https://tailwindcss.com/):** Para el estilizado rápido y sistema de diseño responsivo.
* **TypeScript:** Para garantizar la seguridad de tipos y escalabilidad del código.
* **i18n:** Implementación de internacionalización (Español/Inglés) mediante enrutamiento dinámico.

## Pasos de Instalación :checkered_flag:

Si deseas correr este proyecto localmente para ver cómo está construido:

1.  **Clona el repositorio**
    ```bash
    git clone [https://github.com/TatoNaranjo/portfolio](https://github.com/TatoNaranjo/portfolio)
    cd tato-portfolio
    ```

2.  **Instala las dependencias**
    Necesitas tener Node.js instalado.
    ```bash
    npm install
    ```

3.  **Inicia el servidor de desarrollo**
    Esto iniciará el servidor local de Astro.
    ```bash
    npm run dev
    ```
    * El sitio estará disponible en `http://localhost:4321`.

4.  **Compilar para producción (Opcional)**
    Para generar la carpeta `dist` optimizada:
    ```bash
    npm run build
    ```

## Características Clave :construction:
-   **Mobile-First Design:** Diseño responsivo que escala desde dispositivos móviles hasta pantallas de escritorio.
-   **Internacionalización (i18n):** Soporte nativo para Inglés y Español.
-   **Componentes Interactivos:** Uso de React para modales de proyectos y navegación móvil.
-   **Optimización de Imágenes:** Uso de `astro:assets` para la carga eficiente de medios.
-   **Despliegue Continuo:** Configurado con Netlify Adapter para CI/CD automático.
